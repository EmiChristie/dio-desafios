# Desafio Construindo seu Primeiro Projeto Lógico de Banco de Dados

### O que foi realizado?
Conforme a especificação do desafio, implementei o banco de dados Ecommerce, refinei o esquema e a construção das tabelas e realizei as seguintes melhorias:

- Cliente PJ e PF – Uma conta pode ser PJ ou PF, mas não pode ter as duas informações;
- Pagamento – Pode ter cadastrado mais de uma forma de pagamento;
- Entrega – Possui status e código de rastreio;

### Como foi realizado?
Na pasta "bd_inicial", está uma imagem do esquema final do modelo de banco de dados apresentado no desafio. Além dele, estão scripts SQL que denotam a construção do BD, a inserção de dados, as queries mostradas no desafio e queries adicionais que compõem minha participação.

Já na pasta "bd_refinado" está a parte do desafio que diz respeito à implementação de novas entidades, assim como aprimoramento do que já existia antes. Além das melhorias obrigatórias, também melhorei aspectos como checks para informações, que não existiam antes, assim como a forma de armazenamento do endereço de diversas entidades. O modelo relacional da versão refinada também pode ser encontrado nessa pasta.
