# Desafio Modelando um Dashboard de E-commerce com Power BI Utilizando Fórmulas DAX

###O Desafio
O desafio consistia em criar um esquema estrela a partir da tabela de exemplos disponibilizada pelo PowerBI, sendo que a tabela Calendário deveria ser criada especificamente a partir de fórmulas DAX para manipulação de datas.

###O que foi feito
Seguindo as instruções do vídeo e do docs, criei o esquema com a tabela "Fato_Vendas" sendo a tabela central e as demais, as tabelas dimensão. O processo foi feito basicamente através dos mesmos passos da instrutora, interagindo com a interface do PowerBI para manipular a tabela base e transformá-la em tabelas dimensão segmentadas. Por fim, a tabela Calendário foi criada com fórmulas DAX como YEAR, MONTH, DAY e FORMAT para os diferentes níveis de granularidade/hierarquia de datas. O resultado final pode ser visto visualmente no arquivo "esquema.png".
