# Desafio Analisando dados de um Dashboard de Vendas no Power BI

### O desafio
O desafio consistiu em criar um dashboard de vendas com os dados de exemplo do Power BI, a fim de explorar as opções da ferramenta e ilustrar como dashboards reais são produzidos.

### O que foi realizado
As três páginas de visualizações foram criadas conforme o enunciado do desafio, replicando as ações da instrutora na plataforma. Na pasta, encontram-se o arquivo Power BI e o relatório estático em PDF.

### Como foi realizado
Ao longo das aulas, criei as páginas junto à instrutora (portanto, não usei o código base disponibilizado no desafio, visto que eu já possuía as duas primeiras páginas, construídas manualmente). Para a criação da terceira página, segui os tipos de gráfico e mudanças de visual sugeridas pela instrutora, através da interface gráfica do Power BI.
