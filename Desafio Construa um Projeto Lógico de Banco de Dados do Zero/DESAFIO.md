# Desafio Construa um Projeto Lógico de Banco de Dados do Zero

### O que foi realizado?
Seguindo as especificações do desafio, foi desenvolvida do zero toda a modelagem lógica e física de um banco de dados para uma oficina, incluindo:

- Criação do esquema relacional, contendo entidades como clientes, funcionários, fornecedores, veículos, ordens de serviço, etc;
- Inserção de valores iniciais para popular o banco e permitir testes;
- Criação de checks, uniques, fks nomeadas e uso de enums para garantir a integridade dos dados;
- Desenvolvimento de consultas SQL simples e avançadas.

### Como foi realizado?
Na pasta do desafio, intitulada "Desafio Construindo um Projeto Lógico de Banco de Dados do Zero", encontram-se os seguintes arquivos:

- **criacao_bd**  
  Script SQL contendo a criação completa do banco de dados da oficina.

- **insercao_dados**  
  Conjunto de inserts usados para popular todas as tabelas com dados, permitindo testar a aplicação do modelo.

- **consultas**  
  Documento com todas as queries desenvolvidas, cobrindo desde consultas simples até análises complexas envolvendo diversas tabelas.

- **oficina_desafio_esquema**  
  Arquivo contendo o esquema relacional.
